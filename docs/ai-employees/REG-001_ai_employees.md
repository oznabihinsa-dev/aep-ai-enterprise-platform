# REG-001 — Реестр AI-сотрудников

Код: REG-001  
Версия: 0.1-alpha  
Статус: draft  
Дата: 2026-06-29  
Связанные документы: AI-001, STD-001, BP-001  

## 1. Назначение

Реестр AI-сотрудников фиксирует состав цифровой организации AI-ОФИС.

AI-сотрудник считается существующим в системе только после внесения в настоящий реестр.

## 2. Статусы AI-сотрудников

| Статус | Значение |
|---|---|
| Проектируется | Роль определена, но сотрудник еще не настроен |
| Тестируется | Сотрудник создан и проходит проверку на реальных задачах |
| Работает | Сотрудник используется в рабочих процессах |
| Требует доработки | Обнаружены ошибки или неполная настройка |
| Архивирован | Сотрудник больше не используется |

## 4. Правила изменения реестра

1. Новый AI-сотрудник добавляется только после определения роли.
2. Для каждого AI-сотрудника указывается подразделение.
3. Один AI-сотрудник не должен совмещать несвязанные функции.
4. Если функции пересекаются, решение принимает Архитектор системы.
5. После тестирования статус сотрудника обновляется.
6. Изменения фиксируются через commit в GitHub.
