# AEP — AI Enterprise Platform

AEP — AI Enterprise Platform — это архитектура цифровой организации на базе AI-сотрудников, бизнес-процессов, базы знаний, стандартов и автоматизаций.

## Назначение

Проект предназначен для построения управляемой AI-системы, которая помогает выполнять рабочие задачи по компаниям:

- ООО «ТЕПЛО»
- ООО «ВЭЛС»
- ООО «ДВТК»
- ООО «КАСКАД»

## Принцип архитектуры

- Google Drive — архив рабочих документов и база исходных файлов.
- GitHub — репозиторий архитектуры, стандартов, промптов, маршрутов и автоматизаций.
- ChatGPT Projects — рабочие кабинеты AI-сотрудников.
- n8n — слой автоматизации процессов.

## Структура репозитория

```text
docs/                 Документация системы
prompts/              Инструкции и промпты AI-сотрудников
workflows/            Бизнес-процессы, схемы и n8n-сценарии
templates/            Шаблоны документов, таблиц и чек-листов
scripts/              Скрипты автоматизации
dashboard/            Панель управления системой
```
